# Objectif:
L’objectif de ce mini projet est de fournir un outil capable de protéger un serveur
contre les attaques abusives sur Internet, par bloquer automatiquement les adresses IP
malveillantes.

# How to use

Run 
```
sudo ./script.sh 
```
to check script parameters
